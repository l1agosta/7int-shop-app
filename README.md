# Start guide:

You need postgresql + node.js on your PC

Maybe AFTER all aplication will use on docker containers

* FRONTEND dev build:
```bash
1 cd ./frontend
2 npm install
3. npm run dev
4. http://localhost:5173/
```

* BACKEND:
```bash
1. cd ./backend
2. npm install
3. npm run server
4. http://localhost:PORT
```
